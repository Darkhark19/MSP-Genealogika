# MSP-Genealogika

## A instalar:

- Intalar o node
- Instalar o yarn ($npm install --global yarn)

### Correr o backend

- $cd src
- $yarn install
- $yarn dev
  Após este último o servidor abre na porta 4000

### Correr o Frontend

- $cd genealogika-template
- $yarn install
- $yarn dev
